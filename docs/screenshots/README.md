# Screenshots

This directory contains screenshots for the README and documentation.

## Required Screenshots

- `landing.png` — Landing page screenshot
- `dashboard.png` — Dashboard overview
- `notifications.png` — Notifications list view
- `apps.png` — Connected apps page
- `connect.png` — Integration guide
- `settings.png` — Settings page
- `dark-mode.png` — Dark mode showcase

## Guidelines

- **Resolution:** 1920x1080 or 2x for retina
- **Format:** PNG with transparency where appropriate
- **Naming:** Use lowercase with hyphens

## How to Capture

1. Use browser DevTools for consistent viewport
2. Set viewport to 1440x900 for standard shots
3. Use a screenshot extension for full-page captures
4. Optimize images before committing (use TinyPNG or similar)
